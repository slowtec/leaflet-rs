Leaflet-rs SVG and Canvas Renderer with Multiple Panes Example

This example demonstrates how to use multiple panes with different renderers in leaflet-rs. Each map contains two separate panes, each using its own renderer instance with different styling options.

Panes allow you to organize map layers into different rendering contexts, which is useful for layering content with different visual properties or performance characteristics.

Example 1: SVG Map with Multiple Panes

This map demonstrates two SVG renderers operating on separate panes with different styling configurations:

Pane 1 (Red SVG Renderer)

Red styling with 4px weight, 80% opacity, light red fill. Shapes are positioned at base coordinates.

Pane 2 (Blue SVG Renderer)

Blue styling with 2px weight, 90% opacity, light blue fill. Shapes are slightly offset to show layering.

// Create first SVG renderer for pane1
let svg_options_1 = SvgOptions::default();
svg_options_1.set_pane("pane1".to_string());
svg_options_1.set_color("#ff0000".to_string());
let svg_renderer_1 = Svg::with_options(&svg_options_1);

// Create second SVG renderer for pane2
let svg_options_2 = SvgOptions::default();
svg_options_2.set_pane("pane2".to_string());
svg_options_2.set_color("#0000ff".to_string());
let svg_renderer_2 = Svg::with_options(&svg_options_2);
Example 2: Canvas Map with Multiple Panes

This map demonstrates two Canvas renderers operating on separate panes with different performance and styling configurations:

Canvas Pane 1 (Green Canvas Renderer)

Green styling with 3px weight, optimized for precision with low tolerance and padding.

Canvas Pane 2 (Purple Canvas Renderer)

Purple styling with 5px weight, optimized for performance with higher tolerance and padding.

// Create first Canvas renderer for canvas-pane1
let canvas_options_1 = CanvasOptions::default();
canvas_options_1.set_pane("canvas-pane1".to_string());
canvas_options_1.set_color("#00aa00".to_string());
let canvas_renderer_1 = Canvas::with_options(&canvas_options_1);

// Create second Canvas renderer for canvas-pane2
let canvas_options_2 = CanvasOptions::default();
canvas_options_2.set_pane("canvas-pane2".to_string());
canvas_options_2.set_color("#aa00aa".to_string());
let canvas_renderer_2 = Canvas::with_options(&canvas_options_2);
